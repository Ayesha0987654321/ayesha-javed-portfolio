## 📖 About

This is my personal developer portfolio — a single-page website that showcases my skills, projects, education, and certifications. Built entirely with vanilla HTML, CSS, and JavaScript, it is fully optimized for all screen sizes and designed to leave a lasting impression on anyone who visits.

---

## ✨ Features

- 🎨 **Luxury Dark Gold Theme** — Deep midnight background with warm gold accents and a subtle dot-grid texture
- ⌨️ **Typewriter Animation** — Hero section cycles through professional titles automatically
- 🖱️ **Custom Cursor** — Smooth animated gold dot + ring cursor for desktop
- 📊 **Animated Skill Bars** — Fill up smoothly when scrolled into view
- 🔍 **Scroll Reveal Animations** — Every section fades in gracefully as you scroll
- 📄 **Embedded Certificates** — Click any certification card to view the actual PDF in a modal popup
- 🫧 **Morphing Avatar** — Animated blob shape with smooth CSS keyframe animation
- 📱 **Fully Responsive** — Perfectly optimized for mobile, tablet, and desktop
- 📬 **Contact Form** — With success feedback on submission
- 🔗 **Live Project Links** — Direct links to live demo and GitHub repository

## 🚀 Getting Started

No installation or dependencies required. To run locally:

**1. Clone the repository:**

```bash
git clone https://github.com/Ayesha0987654321/ayesha-javed-portfolio.git
```

**2. Navigate into the folder:**

```bash
cd ayesha-javed-portfolio
```

**3. Open in your browser:**

Simply open `index.html` in any modern browser, or use the **Live Server** extension in VS Code for the best experience:

```
Right-click index.html → Open with Live Server
```

No dependencies to install. That's it! 🎉

## 🌐 Deployment

This portfolio is deployed using **GitHub Pages** for free hosting.

**To deploy your own fork:**

1. Fork this repository
2. Go to **Settings → Pages**
3. Set **Branch: main** → Click **Save**
4. Your site will be live at `https://your-username.github.io/ayesha-javed-portfolio`
